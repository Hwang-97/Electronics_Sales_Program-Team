# 📖 Introduction
### 1. 프로젝트 개요
* 해당 프로젝트는 순수Java를 활용한 console기반 프로젝트이며 전자제품 쇼핑몰을 구현하였습니다.
* 쇼핑몰 활용시 필요한 기초적인 기능을 모두 구현하였으며 관리자 페이지를 추가해서 회원 관리 상품 관리 통계등 관리자가 모든 데이터를 CRUD 가능하게 구현하였습니다.
