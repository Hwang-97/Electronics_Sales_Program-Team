# Java Console Team Project
해당 repo는 2주간 진행된 프로젝트 이며 순수 java기반의 Console 프로젝트 입니다.

팀원은 총 5명으로 진행하였습니다.

프로젝트 구성 순서는

1. 요구분석서 작성
2. 순서도 작성 
3. 화면설계
4. 데이터 설계
5. 코드구현
6. 오류수정
