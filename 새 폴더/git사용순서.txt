
git --version 으로 설치가 되었나 확인

해당 폴더에서 
git clone https://github.com/Hwang-97/Java_5team_Project.git

생성된 폴더로 진입 후

git config --global user.name "입력"

git config --global user.email "가입한 이메일"

내부파일 수정 후 

git status => 수정된 내용 확인 빨간색으로 표시됨

git add * => *은 모든파일 이라는 뜻 파일을 선택하셔도 돼요

git status => 모두 추가 되었나 확인 파란색으로 표시됨

git commit -m "커밋 메시지 입력" 커밋 메시지는 나중에 프로그램 오류시 업로드 지점 확인용으로 사용됨(되돌리기)

git push

이제 https://github.com/Hwang-97/Java_5team_Project 확인해 보시면 아마 추가되어있을거예요
